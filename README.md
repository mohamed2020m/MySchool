## Installation

1. Clone the repo.

```sh
  git clone https://github.com/mohamed2020m/school-web.git
```

2. Install NPM packages

```sh
  yarn install
```

## Usage

Run the project in development

```npm
  yarn dev
```

Build the project

```npm
  yarn build
```
